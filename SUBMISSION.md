# Devfolio Submission

## Project Name
**VeilTrader** - Privacy-First Autonomous AI Trading Agent

## Tagline
Build sovereign AI agents that trade autonomously while keeping user data private.

## Description

VeilTrader is an autonomous AI trading agent built for the Ethereum ecosystem. It uses ERC-8004 identity standards, executes trades via Uniswap V3, and maintains complete privacy through Lit Protocol encryption.

### Key Features

**🤖 Autonomous Trading**
- AI-powered decision making with Bankr LLM Gateway
- Multi-model support (Bankr, Gemini, DeepSeek)
- Real-time market analysis and sentiment
- Automatic trade execution on Base

**🔐 Privacy-First**
- Lit Protocol for encrypted secrets
- Programmable trading keys
- Conditional access control
- Zero data leakage

**⛓️ On-Chain Verification**
- ERC-8004 identity registration
- Trade receipts stored on-chain
- Reputation feedback system
- Full auditability

**💰 Yield Optimization**
- Slice integration for auto-yield
- stETH treasury management
- Multi-protocol deployment

### Tech Stack
- **Blockchain**: Base Sepolia
- **Smart Contract**: Solidity (Foundry)
- **AI**: Bankr LLM Gateway, Gemini, DeepSeek
- **DEX**: Uniswap V3
- **Identity**: ERC-8004, Self.xyz
- **Privacy**: Lit Protocol
- **Frontend**: Vanilla JS + WebSocket
