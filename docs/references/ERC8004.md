```

agentId: ERC-721 tokenId
```

```

```

## Registration Steps

1. Prepare JSON registration file
2. Upload to IPFS
3. Call `identityRegistry.register("ipfs://...", "")`
4. Get ERC-721 tokenId as agentId
5. Add `.well-known/agent-registration.json` for domain verification

## Reputation Registry

### Giving Feedback

```solidity
reputationRegistry.giveFeedback(
    agentId,
    9977,           // value (99.77% uptime)
    2,              // decimals
    "uptime",       // tag1
    "30days",       // tag2
    "https://agent.example.com/api",
    "ipfs://QmDetails",
    keccak256(data)
);
```

### Querying Reputation

```solidity
(uint64 count, int128 value, uint8 decimals) = 
    reputationRegistry.getSummary(
        agentId,
        trustedClients,
        "uptime",
        "30days"
    );
```
